# Manual Verification Checklist

1. Launch Codex Desktop and confirm you are logged in.
2. Run this app and capture the current account with a label.
3. Log in to a different Codex account in Codex Desktop.
4. Capture the second account.
5. Switch back to the first snapshot.
6. Confirm Codex Desktop restarts and the account identity matches the chosen snapshot.
7. Switch to the second snapshot.
8. Confirm the status summary updates.
9. Trigger a failure case manually if desired by corrupting a test snapshot and verifying rollback keeps the previous live account.
