# v0.1.1

## 更新内容

- 新增本地快照删除功能，并在账号池中显示 `5h reset` 与 `Weekly reset` 时间
- 账号池支持按实际可用额度排序：当前账号固定置顶，其余账号按 `min(5h 剩余, 周剩余)` 从高到低排列
- 兼容旧版本本地快照，并在遇到不完整快照目录时继续加载其他可用账号
- Windows 启动链路改为优先使用 Codex 商店版 `AppID`，并在失败时回退到 exe 启动
- Electron `sessionData` 迁移到本地缓存目录，减少 Chromium cache 相关权限报错
- 收紧账号池布局，缩窄 `Action` 面板和 `Switch/Delete` 按钮，并提亮 reset 时间

## 下载说明

- 想正常安装使用，请下载 `Setup` 安装版
- 想免安装直接运行，请下载 `Portable` 绿色便携版

## 当前说明

- 当前正式支持 Windows
- 如果本机没有真实 Codex 登录态，账号和额度数据会显示为空或不可用
